# Sasha Andrews — Portfolio

Personal UX/UI design portfolio site, built from Figma designs.

## 🚀 Deploy to GitHub Pages

1. Create a new repo on GitHub (e.g. `sasha-portfolio`)
2. Push these files:
```bash
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/sasha-portfolio.git
git push -u origin main
```
3. Go to **Settings → Pages → Source: Deploy from branch → main / root**
4. Your site will be live at `https://YOUR_USERNAME.github.io/sasha-portfolio/`
